Programming language
is a vocabulary and a set of grammatical rules to instruct a computer or computing device to perform specific tasks. The word programming language usually refers to high-level languages such as Visual Basic, BASIC, C, C++, COBOL, Python, Java, FORTRAN, Ada, and Pascal. Every programming language has a specific set of keywords (words it understands) and a special syntax for organizing program instructions. Hundreds of different programming languages have been created and more are being created each year. Most programming languages are written in an imperative form while other languages use the declarative form.
